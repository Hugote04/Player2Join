##🎮 Player2Join | Video Game Collective & Manager
📝 Descripción del Proyecto
Player2Join es una Single Page Application (SPA) de alto rendimiento diseñada con Angular 21. La plataforma permite a los usuarios explorar la inmensa base de datos de RAWG, gestionar su propia biblioteca de juegos y conectar con el "Player 2" mediante un sistema de autenticación seguro.

Este proyecto cubre los objetivos de los RA6, RA7, RA8 y RA9 del módulo de Desarrollo Web en Entorno Servidor.

🚀 Stack Tecnológico
Frontend: Angular 21 (Arquitectura de Standalone Components).

Gestión de Estado: Uso de Angular Signals para una reactividad moderna (RA8).

Backend & Auth: Firebase (Authentication para JWT y Firestore para base de datos).

API Externa: RAWG Video Games Database para el catálogo de juegos.

Seguridad: Guardias de ruta (AuthGuard), Interceptores HTTP y validación de roles.
