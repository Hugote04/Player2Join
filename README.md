🎮 Player2Join | Video Game Collective & Manager
Este proyecto es una Single Page Application (SPA) desarrollada con Angular 21 para el proyecto final de la asignatura Desarrollo Web en Entorno Servidor (DWES). La aplicación permite explorar el catálogo de RAWG API, autenticarse y gestionar una colección personalizada de videojuegos.





🚀 Stack Tecnológico (RA8)

Frontend: Angular 21 con Standalone Components.



Gestión de Estado: Angular Signals para reactividad moderna.


Backend & Auth: Firebase (Authentication para JWT y Firestore para base de datos).

API Externa: RAWG Video Games Database para el catálogo de juegos.


Seguridad: Guardias de ruta (AuthGuard), Interceptores HTTP y validación de roles.
